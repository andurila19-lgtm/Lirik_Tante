# 🎵 Tante Lirik - Python Script

skrip Python sederhana yang menampilkan lirik lagu "Tante" dengan efek mengetik (*typing effect*) dan pewarnaan teks ANSI.

## ✨ Fitur
- **Efek Mengetik**: Lirik muncul karakter demi karakter layaknya sedang diketik secara real-time.
- **Warna Teks**: Menggunakan kode warna ANSI untuk membedakan baris-baris lirik.
- **Timing Akurat**: Jeda antar baris dan karakter disesuaikan agar pas dengan nuansa lagu.

## 🛠️ Prasyarat
Untuk menjalankan skrip ini, Anda hanya perlu menginstal **Python 3** di sistem Anda.

## 🚀 Cara Menjalankan
1. Pastikan Anda berada di direktori proyek:
   ```powershell
   cd d:\Tantee
   ```
2. Jalankan skrip menggunakan Python:
   ```powershell
   python tante.py
   ```

## 📝 Contoh Tampilan
```text
~TAANTEEEEEEEE~
 Temanku semua pada jahat tante
 aku lagi susah mereka gak ada
 coba kalau lagi jayaaa
 ...
```
